■ Sammanfattning av appdata

När du använder våra PlayStation-appar samlar vi in information om appen, enheten den är installerad på och hur du använder den. Vi kallar denna information för appdata.

Vi samlar alltid in information som behövs för att appen ska kunna köras. Dessa uppgifter kallas begränsade data och omfattar följande typer av information:

  • det unika namnet på enheten som appen körs på (ofta en lång rad bokstäver och siffror)
  • inställningar som du har angett i appen (som datum, region och språk)
  • din internetanslutning
  • hur lång tid det tar att ladda ned information på appen (så att vi vet om något går snett och kan fixa det)

En del information samlas bara in om du tillåter det. Denna information kallas fullständiga data. Detta innefattar mer detaljerad information om dig, till exempel sidvisningar och klick när appen används.

Vi använder den information appen skickar för att:

  • hjälpa dig när du ber om det.
  • se till att våra appar och produkter fungerar bättre. 
  • försäkra oss om att användarna följer våra regler och gällande lagar (så att PlayStation Network är roligt för alla).
  • få en bättre förståelse för vad våra kunder gillar. 
  • hjälpa oss att utveckla nya produkter.

Om du har ett konto för PlayStation Network lägger vi till den information enheten skickar till den information vi redan har om dig. 

När ett barnkonto använder denna app samlas endast begränsade data in.

I vår sekretesspolicy kan du läsa mer om vad vi gör med den insamlade informationen, vem vi delar den med, hur du bestämmer vad vi får göra med den och vilka rättigheter du har. Du kan läsa sekretesspolicyn på www.playstation.com/legal/privacy-policy. 

■Förklaring av fullständiga appdata

Vad är appdata?

Om du använder en av våra PlayStation-appar kan vi använda data från och/eller lagra data på mobilen, surfplattan, datorn och/eller andra enheter du använder.

Det finns två slags appdata:

(1) Begränsade data

Våra servrar måste skicka, ta emot och logga följande begränsade data för att leverera våra tjänster:

  • Innehållsdata: Video, ljud, grafik, text, spelnivå, kontoinformation, annonser och annat innehåll som krävs och laddas ned för de funktioner, spel, appar och webbplatser du använder online eller laddas upp när du delar sådant innehåll via sociala funktioner och kommunikationsfunktioner i våra onlinetjänster. Om du spelar onlinespel innefattar det här alla dina interaktioner online (t.ex. karaktärens rörelser) beroende på vad som krävs för att spegla dessa interaktioner på våra spelservrar.
  • Licensdata: Information om visning av och interaktion med licensierat innehåll (t.ex. videor) som krävs för att producera anonymiserade, sammanfattande rapporter för att betala licensgivare för det innehållet.
  • Nätverksdata: Information om din enhets nätverksanslutningar (som mobil-, Ethernet-, Wi-Fi- och Bluetooth-anslutningar), inklusive IP-adress samt anslutningarnas antal, krav och hastigheter.
  • Konfigurationsdata: Information om din enhets konfiguration, t.ex. tillverkare, modell, antal processorer, skärmstorlek och upplösning, datum, region, språk, nationella inställningar och funktioner.
  • Data om ansluten maskinvara: Information om maskinvara som är ansluten till din enhet, t.ex. spelkontroller och annan kringutrustning.
  • Feldata: Information om felkoder och grundläggande data om när och var eventuella fel inträffar (ett fel kan tyda på ett problem som behöver åtgärdas).
  • Data om laddningstid: Grundläggande information om hur lång tid det tar att ladda ned och upp data till din enhet (lång tid kan tyda på att något är fel).
  • Sessionsdata och diagnostiska data: Mer detaljerad navigeringsinformation, inklusive data om visning av innehåll och annonssidor, klick, exponeringar och videoströmmar liksom fel vid start av appar/spel och laddningstider. Om det krävs för att åtgärda problem med en onlinetjänst som användaren begärt kan den här informationen kopplas till kontoidentifierare för att lösa problemet.
  • Säkerhetsdata: Information som krävs för att upptäcka och undersöka skadlig, obehörig eller bedräglig aktivitet.
  • Juridiska data: Information som krävs enligt gällande lagstiftning och för regelefterlevnad.

 

(2) Fullständiga data:

Vi kan också komma att skicka, ta emot och logga ytterligare och mer detaljerad information om konfiguration av appen, mobilen, surfplattan eller någon annan enhet och vilken programvara som är installerad på den. Fullständiga data kan inbegripa mer detaljerad information om sidvisningar, klick, exponeringar och videoströmmar. De kan också inbegripa data om hur du använder appen eller enheten när du inte är ansluten till en onlinetjänst (den informationen laddas då upp när appen eller enheten är ansluten till internet nästa gång).

Exempel:

  • Om du spelar något av våra spel samlar vi inte bara in nödvändiga begränsade onlinedata (som att du har behörighet att starta spelet, ladda ned en nivå och använda våra spelservrar), utan vi kan också använda fullständiga data om din spelinteraktion. Det kan gälla när och hur du använder spelet, vilka delar av spelnivån du går till, vilka vapen du använder och vilka karaktärer du interagerar med.
  • Om du besöker PlayStation Store samlar vi inte bara in nödvändiga begränsade onlinedata (som nedladdning av innehåll och transaktionshantering), utan vi kan också lagra och använda beständiga identifierare på din app eller enhet och mer detaljerade data om din interaktion med butiken, till exempel vart du rullar i olika stycken och vilka objekt du hovrar över.
  • Om du använder någon av våra appar samlar vi inte bara in nödvändiga fullständiga onlinedata (som att du har behörighet att ladda ned innehåll i appen från våra servrar), utan vi kan också använda ytterligare data om din interaktion med appen och enheten. Det här kan inbegripa hur du navigerar i olika delar av appen och hur länge du tittar på olika funktioner.

 

Appdata överförs vanligen till oss och lagras med en eller flera unika nätverks-, enhets- och/eller kontoidentifierare.

Om du loggar in på ditt konto för PlayStation Network från ett PlayStation-system (t.ex. PlayStation 4, PlayStation 5) kan vi komma att kombinera appdata med data som har samlats in från det systemet. Läs mer i vår sekretesspolicy (www.playstation.com/legal/privacy-policy eller via enhetsinställningarna).

Vad är det inställningarna för appdata styr?

Med inställningarna för appdata kan du styra vilka appdata vi lagrar och använder. Du kan välja: 

  • Begränsade data: Om du väljer den här inställningen är det bara grundläggande data som används och lagras.
  • Fullständiga data: Om du väljer den här inställningen kommer både grundläggande data och fullständiga data att användas och lagras.

 

Inställningarna för appdata är bara associerade med den enhet eller app som de visas på och är inte associerade med ditt konto i PlayStation Network. Du styr användning och lagring av data för respektive enhet (t.ex. PS4, PS5), app och webbläsare som du använder, oberoende av de andra.

Vad använder vi appdata till?

Begränsade data och, om du tillåter det, fullständiga data används så som beskrivs i vår sekretesspolicy (www.playstation.com/legal/privacy-policy).

Det här omfattar användning av appdata för att:

  • tillhandahålla och anpassa våra onlinetjänster
  • ta fram anonym statistik
  • förstå prestanda för appen eller enheten och utveckla förbättringar och nya produkter och tjänster
  • identifiera, undersöka och minimera bedrägligt, obehörigt eller skadligt beteende
  • beroende på vilka inställningar du har, leverera, målinrikta och anpassa reklam, kommersiella rekommendationer och marknadsföringskommunikation
  • genomföra revision samt följa gällande regler och lagar (till exempel vid anspråksförfarande eller andra aktiviteter som måste uppfylla viss lagstiftning).
